# reports/

This directory stores all output artifacts generated by the pipeline — visualizations, model evaluation figures, and saved model files.

## Subdirectories

| Directory | Description |
|-----------|-------------|
| `figures/` | Charts and plots generated during EDA, model evaluation, and ML output visualization. Saved as PNG or SVG. |
| `models/` | Saved trained model artifacts (Keras `.h5` files for LSTM, Pickle `.pkl` for scikit-learn models). Excluded from git via `.gitignore`. |

## Figure Naming Convention

Use the following naming pattern for all figures saved to `reports/figures/`:

```
<model>_<description>_<date>.png

Examples:
  lstm_actual_vs_predicted_20260621.png
  kmeans_elbow_curve_20260621.png
  kmeans_pca_scatter_cluster_20260621.png
  rf_feature_importance_20260621.png
  rf_roc_curve_20260621.png
```
